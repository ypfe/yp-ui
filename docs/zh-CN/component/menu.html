# 菜单

yp-ui菜单基于element-plus菜单， 增加了统一的菜单 样式

通过类名yp-aside-dark和yp-aside-white可以实现菜单的颜色模式改变

统一菜单颜色

$menu-bg-color: #505050; // 菜单背景色
$menu-text-color:#ccc; // 菜单文字颜色
$menu-hover-color:#fff; // hover/active高亮颜色
$subMenu-bg-color:#333333;// 子菜单背景色
$subMenu-bg-hover-color:#0B0B0C; // 子菜单hover/激活背景色
$menu-bg-color-white: #fff; // 菜单背景色
$menu-text-color-white:#333; // 菜单文字颜色
$menu-hover-color-white:#fff; // hover/active高亮颜色
$subMenu-bg-color-white:#2E59A7;// 子菜单背景色
$subMenu-bg-hover-color-white:#D5DEED; // 子菜单hover/激活背景色

应用

<el-menu
    class="yp-aside-dark"
    router
    :mode="mode"
    :default-active="defaultActivePath"
    :collapse="isCollapsed"
    >
    <sub-menu-com v-for="(menu, i) in permissionRoutes" :key="i" :item="menu" />
  </el-menu>
